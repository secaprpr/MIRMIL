# Documentation Index

This directory is an archive of the MIL research process.

- `MIRMILV1_METHOD.md`: paper-level method description for the archived
  MIR-MIL v1 line, including the neural measure state, adaptive multiscale
  potential, accepted moment-token readout, training objective, influence
  response, limitations, and ablation boundary.
- `MIR-MIL_IMPLEMENTATION.md`: implementation and experiment history for
  MIR-MIL, including earlier PANDA-focused verification and follow-ups.
- `WANDB_EXPERIMENT_TRACKING_DESIGN.md`: experiment tracking design used for
  reproducible metric/config logging.
- `PANDA_FEATURE_PIPELINE_IMPLEMENTATION.md`: PANDA WSI tissue segmentation,
  patching, stitching, and feature extraction workflow.
- `PANDA_BASELINE_RESULTS.md`: PANDA baseline result summary.
- `BRACS_BASELINE_RESULTS.md`: BRACS baseline result summary.

- `OT-MIL.md`: original idea and theoretical design notes. It describes the
  intended minimal sufficient diagnostic submeasure framework, not the final
  empirically supported claim.
- `EXPERIMENT_REPORT_2026-06-10.md`: complete experiment history, commands,
  provenance, ablations, failures, and final benchmark results.
- `../PROJECT_CLOSEOUT.md`: authoritative final conclusion and project status.

For a new reader, use this order:

1. `../README.md`
2. `MIRMILV1_METHOD.md`
3. `MIR-MIL_IMPLEMENTATION.md`
4. `WANDB_EXPERIMENT_TRACKING_DESIGN.md`
5. `PANDA_FEATURE_PIPELINE_IMPLEMENTATION.md`
6. `PANDA_BASELINE_RESULTS.md`
7. `BRACS_BASELINE_RESULTS.md`
8. `EXPERIMENT_REPORT_2026-06-10.md`
9. `OT-MIL.md`

The long documents are intentionally retained rather than rewritten into a
clean success narrative. Negative results and abandoned variants are part of
the reproducibility record.
